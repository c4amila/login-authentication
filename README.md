# 🔐 Sistema de Autenticação de Usuários

Este é um projeto que simula um sistema simples de autenticação de usuários, utilizando a arquitetura em camadas, persistência de dados, segurança e envio de e-mails.

## Funcionalidades
- Cadastro de usuário com validação por código
- Login com controle de tentativas de senha
- Bloqueio após 5 tentativas inválidas
- Recuperação de senha via código
- Logout
